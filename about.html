Common HTML Text Formatting Tags
Tag Description Example Output

<i> Makes text italic (visual only) This is italic text

<u> Underlines the text This is underlined text
<mark> Highlights the text with a yellow background This is highlighted text
<s> Strikes through the text (shows deleted content) This text is struck through
<small> Makes the text smaller than normal This is small text
