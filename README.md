# Monia Nijimbere — Personal Profile Website

A personal profile website built as part of an HTML Fundamentals Assignment. The site is built using HTML and CSS only.

## Pages

- **Home** — Introduction, what I love, and travel photography
- **About** — My photo, biography, and academic courses
- **Contact** — My contact details and a message form

## What Was Built

- Structured HTML pages with proper semantic elements
- Navigation bar linking all pages together with tooltips
- Ordered and unordered lists
- Images with alt text, width, and height attributes
- An image that acts as a clickable hyperlink
- A data table with caption, thead, and tbody
- A contact form with text input, email input, radio buttons, checkboxes, labels, and a submit button
- A text formatting reference table showing common HTML tags
- An external CSS file for styling

## Technologies Used

- HTML5
- CSS3
- Google Fonts

## How To View

1. Download or clone the repository
2. Open `index.html` in any web browser
3. Navigate between pages using the navigation bar
